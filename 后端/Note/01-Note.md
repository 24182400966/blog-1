# Note

## 001 IDEA 常用快捷键

| 快捷键           | 描述         |
| :--------------- | :----------- |
| Ctrl+Shift+Alt+N | 查找类名     |
| Ctrl+Alt+O       | 移除未使用包 |
| Ctrl+Shift+Enter | 换行         |
| Ctrl+Shift+U     | 大小写转换   |
| Ctrl+Alt+T       | 语句快捷键   |
