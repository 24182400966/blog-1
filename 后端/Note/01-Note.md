## SSM 概念

SSM：Spring + Spring MVC + MyBatis

- Spring 是一个轻量级的控制反转（IoC）和面向切面（AOP）的容器框架。
- SpringMVC 分离了控制器、模型对象、分派器以及处理程序对象的角色，这种分离让它们更容易进行定制。
- MyBatis 是一个支持普通 SQL 查询，存储过程和高级映射的优秀持久层框架。
